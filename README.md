## 앱의용도
대한민국에서 자선단체 정보를 빠르게 확인 가능하다.

## 기능
  - 시도/시군구 전체 인구 대비 저소득한부모가구 비율을 색으로 확인 할 수 있다.
  - 시군구 지역을 누르면 해당 지역의 자선단체 정보를 확인가능하다.
